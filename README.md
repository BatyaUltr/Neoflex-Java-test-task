# Тестовое задание для учебного центра Neoflex "Калькулятор отпускных"

## Микросервис на SpringBoot + Java 11 c одним API: GET "/calculate"

### Минимальные требования:
Приложение принимает твою среднюю зарплату за 12 месяцев и количество дней отпуска - 
отвечает суммой отпускных, которые придут сотруднику.

### Доп. задание:
При запросе также можно указать точные дни ухода в отпуск, тогда должен проводиться 
рассчет отпускных с учётом праздников и выходных.

### Используемые технологии: 
- Java 11;
- Gradle (Groovy);
- Spring Boot;
- Junit (для тестирования).

### Запуск проекта:
- Для построения проекта выполните команду: `./gradlew clean build`;
- Для запуска проекта выполните команду: `./gradlew bootRun`.

### Примеры использования программы
1. Для расчета отпускных без учета выходных и праздников, используя количество 
отпускных дней: 
<localhost:8080/api/vacations/calculate?salaryPerYear=x&vacationDays=y>, 
где x - годовая зарплата, y - количество дней отпуска.

2. Для расчета отпускных с учетом выходных и праздников, используя дату 
начала и конца отпуска:
<localhost:8080/api/vacations/calculate?salaryPerYear=x&startDate=a&endDate=b>, 
где a и b - дата начала и конца отпуска, соответственно, в формате YYYY-MM-dd.

3. Для расчета отпускных с учетом выходных и праздников, используя дату 
начала отпуска и количество отпускных дней:
<localhost:8080/api/vacations/calculate?salaryPerYear=x&startDate=a&vacationDays=y>,
где x - дата начала отпуска в формате YYYY-MM-dd, y - количество дней отпуска.

***Тестирование производилось с помощью Postman и Junit.***
